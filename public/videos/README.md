# Hero background video

Place your loop here as:

**`hero-shopping.mp4`**

Recommendations:

- Short loop (10–30 s), **muted**, fashion retail / boutique / luxury shopping mood
- Encode for web: H.264 `.mp4`, reasonable bitrate (e.g. 720p–1080p)
- The storefront expects this path: `/videos/hero-shopping.mp4`

If the file is missing or fails to load, the hero shows a static gradient fallback.
